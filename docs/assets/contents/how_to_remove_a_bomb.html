Thanks to Mr. Mohamed Felfel's instruction, I eventually managed to remove BOMbs. Here are the steps:
  1. Open the UTF-8 BOM file with vi vi {filename} in terminal
  2. Use command :setlocal nobomb to remove BOM
  3. Save file with :w
  4. Quit file with :q
  5. Optionally, if you check the file in IDE with VCS, it will show no content change but the file format is changed from UTF-8 BOM to UTF-8
To check if there is a BOM, use command :setlocal bomb? in vi. If there is no BOM, it shows nobomb. If there is, it shows bomb
