Try to set netbeans_jdkhome in {NetBeans directory}/etc/netbeans.conf

# Default location of JDK, can be overridden by using --jdkhome : netbeans_jdkhome="..."

P.S. This will work if you use an archive netbeans ide. However, if you just want to install netbeans, I believe an installed JDK would be a prerequisite to start installation wizard of netbeans.
